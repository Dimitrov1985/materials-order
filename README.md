# Заявка на строительные материалы

Консольное приложение на Node.js для формирования заявок на строительные материалы с учётом региона и логики удержания клиента.

## Возможности

- Выбор региона доставки (Санкт-Петербург, Москва, Краснодар)
- Просмотр каталога материалов с актуальными ценами для региона
- Формирование заявки с указанием количества
- Сохранение заявки в JSON-файл
- Логика удержания клиента при отказе:
  - Предложение более дешёвого аналога в той же категории
  - Персональная скидка 5%, если выбранный товар уже самый выгодный

## Установка и запуск

```bash
git clone https://github.com/Dimitrov1985/materials-order.git
cd materials-order
node index.js
```

Требуется Node.js версии 16 и выше.

## Структура проекта

```
├── index.js          # Точка входа
├── catalog.js        # Каталог товаров
└── src/
    ├── regions.js    # Справочник регионов
    ├── ui.js         # Отображение в консоли
    ├── input.js      # Обработка ввода пользователя
    ├── order.js      # Сохранение заявки в JSON
    └── retention.js  # Логика удержания клиента
```

  "region": "МСК",
  "regionName": "Москва",
